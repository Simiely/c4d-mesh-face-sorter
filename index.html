01 / Features

六个功能，一个面板。

打开面板→点击刷新→搞定。所有操作集中在一个对话框，不多不少。

🔄

排序

按面数或存储大小排序，点按切换升降序。快速定位场景中最大的网格体。

👁

孤立显示

点击 O 按钮，只显示选中物体，其余隐藏。专注编辑单个物体的绝佳方式。

▶

选中高亮

列表中选中物体加 ▶ 标记，一目了然。点击行即可在场景中选中对应物体。

🗑

删除空物体

一键清理面数为 0 的空网格体。安全删除——有子级的不删，绝不误伤。

📊

导出报表

导出 Markdown 格式的场景报告，含面数 / 存储统计。方便分享与归档。

🌐

显示全部

一键恢复所有隐藏物体。孤立显示后的完美搭档，快速回到完整场景。

02 / Usage

三步上手，即刻高效。

安装即用，无需配置。界面直观，操作路径短到极致。

01

安装插件

将 mesh_face_sorter.pyp 放入 C4D 插件目录，创建 res/c4d_symbols.h，重启 C4D。

02

打开面板

扩展 → Mesh Face Sorter，点击「刷新」扫描整个场景。所有多边形物体即刻列出。

03

排序与操作

按面数或存储大小排列，点击行选中物体，点 O 孤立编辑，一键导出报表或删除空物体。

03 / Architecture

双层架构，稳固可靠。

数据层与展示层分离，排序算法与对话框逻辑清晰解耦。

排序引擎

面数
递归统计 GetPolygonCount，参数化物体通过缓存获取
存储
按 点 × 24 + 面 × 16 估算内存占用
缓存
参数化物体自动降级到 GetCache 递归统计

对话框系统

引用持有
CommandData 保持对话框引用，防止 GC 回收
RestoreLayout
直接返回 True，避免二次打开崩溃
异常保护
所有回调包裹 try/except，静默容错
