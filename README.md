# C4D Mesh Face Sorter

> Cinema 4D 插件：按面数/顶点数/存储大小排列场景中所有网格体，快速定位高面数模型、批量减面、孤立显示、删除空网格、导出场景报表。

**目标版本：** C4D 2023+（兼容 2024/2025/2026）

---

## 功能一览

| 功能 | 说明 |
|---|---|
| **多维度排序** | 按面数 / 顶点数 / 存储大小排序，升降序切换 |
| **存储大小估算** | 基于点/多边形/UV/顶点色估算内存占用 |
| **减面 Tag** | 批量或单个添加 Polygon Reduction Tag，支持自定义保留比例 |
| **应用减面** | 一键应用选中物体的 Polygon Reduction Tag |
| **孤立显示** | 隐藏其他所有网格体，只看当前选中物体 |
| **删除空网格** | 清理面数为 0 的空网格体（带确认对话框） |
| **清理未使用数据** | 清理场景中未使用的材质 |
| **导出 md 报表** | 导出 Markdown 表格，含统计信息和每物体详情 |
| **扫描进度** | 大场景扫描时显示百分比进度条 |

## 安装

### 方式一：手动复制（推荐）

1. 下载本仓库内容
2. 将 `c4d-mesh-face-sorter` 文件夹复制到 C4D 的插件目录：
   - **Windows:** `C:\Program Files\Maxon Cinema 4D 202X\plugins\`
   - **macOS:** `/Applications/Maxon Cinema 4D 202X/plugins/`
   - **用户目录:** `%APPDATA%/Maxon/Cinema 4D 202X/library/plugins/` (Windows) 或 `~/Library/Preferences/Maxon/Cinema 4D 202X/library/plugins/` (macOS)
3. 重启 Cinema 4D

### 方式二：克隆仓库

```bash
git clone https://github.com/Simiely/c4d-mesh-face-sorter.git
```

将文件夹按方式一的步骤复制到 C4D 插件目录。

## 使用方法

### 打开面板

菜单 → **扩展** → **Mesh Face Sorter**

### 面板概览

```
┌──────────────────────────────────────────┐
│ ✓ 扫描完成：12 个网格体（14:23:05）       │  ← 状态区
├──────────────────────────────────────────┤
│ 网格体：12    面数：1.2M                 │  ← 统计区
│ 顶点：600K    存储：45.3 MB               │
├──────────────────────────────────────────┤
│ 排序：[面数 ▼]  [↓↑]                     │  ← 排序切换
├──────────────────────────────────────────┤
│ [刷新列表]  [选中所有网格体]              │  ← 操作按钮
│ [显示全部]  [删除无面]  [清理数据]        │
│ [导出 md 报表]                           │
├──────────────────────────────────────────┤
│ 保留比例：[0.500]                        │  ← 减面控制
│ [  减面 Tag  ] [  应用减面  ]            │
├──────────────────────────────────────────┤
│ 物体名称              │ 面数*│ O │ -    │  ← 物体列表
│ ▶ Body_High           │ 500K │ O │ -    │
│   Head                │ 120K │ O │ -    │
└──────────────────────────────────────────┘
```

### 操作说明

| 操作 | 效果 |
|---|---|
| 点击物体名称 | 选中该物体并设为活动对象 |
| 点击 **[O]** 按钮 | 孤立显示该物体（隐藏其他所有网格体） |
| 点击 **[-]** 按钮 | 给该物体添加 Polygon Reduction Tag |
| 点击 **「刷新列表」** | 重新扫描场景 |
| 切换排序方式 | 面数/顶点/存储大小（瞬时切换，不重扫） |
| 调整比例 | 设置 Tag 的保留比例（0.01~1.0） |
| 点击 **「减面 Tag」** | 给选中物体批量添加 Polygon Reduction Tag |
| 点击 **「应用减面」** | 应用选中物体的 Polygon Reduction Tag |
| 点击 **「清理数据」** | 删除未使用的材质 |

### 典型场景

**找最占资源的模型：** 刷新 → 排序切到「存储大小」→ 第一名即最大

**批量减面：** 选中多个物体 → 调好比例 → 点「减面 Tag」

**导出场景报告：** 点「导出 md 报表」→ 选择路径 → 得到完整 Markdown 文件

**清理脏数据：** 点「删除无面」+「清理数据」

## 性能说明

- **缓存机制：** 面板不每帧重扫，切换排序方式只重排缓存
- **手动刷新：** 增删物体后点「刷新列表」才重新扫描
- **列表限制：** 最多显示 500 个，超出可导出报表查看全部
- **进度提示：** 扫描时显示百分比进度条
